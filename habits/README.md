# habits

the agents in the root and the loops in `loops/` are tools. these are the practices i run *around* the tools — the cheap, daily things that compound and that most people skip because they're not flashy.

a tool makes one task faster. a habit makes every future task faster, because it changes what you carry forward. the agents got better over years; the habits are why i knew what to point them at.

## the habits

| habit | what it is |
|-------|------------|
| [dev journal](dev-journal.md) | write down every mistake + the rule that stops it next time. the single highest-return thing i do. |

each file: what the habit is, how to actually run it (the ritual), a template, and a copy-paste prompt so your agent does the heavy lifting instead of you staring at a blank file.

---

steal these, run them daily, tweak them for how your brain works.
